<h2 id="usage" > 👷 Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/RodrigoSaantos/nlw-return-impulse.git

# Go to folder classroom
$ cd web/

# Install Dependencies
$ yarn
# or
$ npm install

# Run Application
$ yarn dev
# or
$ npm run dev

# Access localhost
http://localhost:3000

# Go to folder purchases
$ cd server/

# Install Dependencies
$ yarn
# or
$ npm install

# Run Application
$ yarn dev
# or
$ npm run dev

# Access localhost
http://localhost:3333/

# Go to folder gateway
$ cd mobile/

# Install Dependencies
$ yarn
# or
$ npm install

# Run Application
$ expo start
```
